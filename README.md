## 📊 Sobre o Projeto
Este projeto realiza uma análise estatística aprofundada dos microdados do ENEM 2023, focando no impacto da renda familiar e outros fatores socioeconômicos no desempenho dos candidatos. O objetivo é transformar dados brutos em insights acionáveis para políticas educacionais e estratégias de inclusão.

## 🚀 Funcionalidades
- **Pipeline POO:** Estrutura robusta utilizando os pilares da Programação Orientada a Objetos.
- **Limpeza e Tratamento:** Processamento de grandes volumes de dados com amostragem estratificada.
- **Análise Estatística:** 
  - Testes de hipótese (ANOVA, Spearman).
  - Modelagem preditiva e inferencial (Regressão OLS).
- **Visualização:** Gráficos avançados (Boxplots, Heatmaps, Violin Plots) para storytelling de dados.
- **Geração de Relatórios:** Exportação automática de insights em Markdown.

## 🛠️ Tecnologias
- **Linguagem:** Python 3.x
- **Bibliotecas:** Pandas, NumPy, Matplotlib, Seaborn, Statsmodels, Scipy.

## 📁 Estrutura do Repositório
- `analise_enem.py`: Script principal com a lógica de processamento e análise.
- `outputs/`: Gráficos e visualizações geradas.
- `relatorio_final.md`: Documento com os principais insights da análise.
- `requirements.txt`: Dependências do projeto.

## 📈 Como Executar
1. Clone o repositório:
   ```bash
   git clone https://github.com/MateusSantos590/Projeto_analise_dados-desigualdade_educacional_no_brasil_ENEM23.git
   ```
2. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
3. Execute a análise:
   ```bash
   python analise_enem.py
   ```
